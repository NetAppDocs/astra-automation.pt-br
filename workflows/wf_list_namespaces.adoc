= Liste os namespaces
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Você pode listar os namespaces disponíveis.

Execute a seguinte chamada de API REST para listar os namespaces.

.Método HTTP e endpoint
Essa chamada de API REST usa o método e o endpoint a seguir.

[cols="1,6"]
|===
| Método HTTP | Caminho 


| OBTER | /accounts/_id/topology/v1/namespaces 
|===
.Curl exemplo: Retorna todos os dados para todos os namespaces
[source, curl]
----
curl --request GET \
--location "https://astra.netapp.io/accounts/$ACCOUNT_ID/topology/v1/namespaces" \
--include \
--header "Accept: */*" \
--header "Authorization: Bearer $API_TOKEN"
----
.Curl exemplo: Retorna nome, estado e ID de cluster para todos os namespaces
[source, curl]
----
curl --request GET \
--location "https://astra.netapp.io/accounts/$ACCOUNT_ID/topology/v1/namespaces?include=name,namespaceState,clusterID" \
--include \
--header "Accept: */*" \
--header "Authorization: Bearer $API_TOKEN"
----
